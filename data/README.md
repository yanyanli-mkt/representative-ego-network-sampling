# Data Directory

This directory holds all raw and pre-computed data files required by the
replication pipeline. The structure after running `01_prepare_networks.py` is
shown below.

---

## Step 1: Download real-world network data

### LastFM-Asia

**Source**: https://snap.stanford.edu/data/feather-lastfm-social.html
(Rozemberczki and Sarkar, 2020)

Download `lastfm_asia.zip`, extract, and place:
- `lastfm_asia_edges.csv` → `data/lastfm/`
- `lastfm_asia_target.csv` → `data/lastfm/`
- `lastfm_asia_features.json` → `data/lastfm/`

### Twitch-Gamer

**Source**: https://snap.stanford.edu/data/twitch.html
(Rozemberczki and Sarkar, 2021)

Download `twitch.zip`, extract, and place:
- `large_twitch_edges.csv` → `data/twitch/`
- `large_twitch_features.csv` → `data/twitch/`

### Pokec

**Source**: https://snap.stanford.edu/data/pokec.html
(Takac and Zabovsky, 2012)

Download and place:
- `soc-pokec-relationships.txt` → `data/pokec/`
- `soc-pokec-profiles.txt` → `data/pokec/`

> **Note**: Pokec has ~1.6M nodes. `01_prepare_networks.py` takes ~2 hours.
> Use `--skip_hop2` for large networks — hop-2 is computed on-the-fly during
> sampling.

## Step 2: Generate network attributes

Run `01_prepare_networks.py` once to compute and save all network attributes:

```bash
# All networks (simulated networks require no download)
python scripts/01_prepare_networks.py

# Skip hop-2 pre-build for large networks (pokec, simulate_base, simulate_sparse)
python scripts/01_prepare_networks.py --skip_hop2
```

This generates for each network:
- `a_degree.csv`, `a_cluster_coef.csv`, `a_eigen_centrality.csv`, `a_nb_degree.csv`
- `hop1_list.pkl` (and `hop2_list.pkl` if not skipped)
- `Z1.csv`, `Z2.csv`, `Z3.csv` (simulated networks only)
- `network_summary.csv`

And the combined `data/table1_network_summary.csv` (Table 1).

---

## Exposure mapping files

`05_causal_inference.py` estimates IPTW exposure probabilities via Monte Carlo
simulation (500,000 draws) and saves them as:

```
output/tables/<network>-Ego-Correction-Expo-Mapping-500sample.csv
```

These are computed automatically on first run (~10 min each). To reuse:

```bash
python scripts/05_causal_inference.py --skip_expo_mapping
```

## File formats

| File | Format | Description |
|------|--------|-------------|
| `a_degree.csv` | Plain text, one value per line | Integer degree for each node |
| `a_cluster_coef.csv` | Plain text, one value per line | Full-precision clustering coefficient |
| `a_eigen_centrality.csv` | Plain text, one value per line | Eigenvector centrality |
| `a_nb_degree.csv` | Plain text, one value per line | Average neighbor degree |
| `hop1_list.pkl` | Python pickle | `dict[int, set[int]]` — node → set of hop-1 neighbors |
| `hop2_list.pkl` | Python pickle | `dict[int, set[int]]` — node → set of hop-2 neighbors |
| `Z1.csv` | Plain text, one value per line | Covariate: rho=-0.7 with degree, scaled [0,10] |
| `Z2.csv` | Plain text, one value per line | Covariate: rho=+0.3 with degree, scaled [0,10] |
| `Z3.csv` | Plain text, one value per line | Covariate: Bernoulli(0.5) |
| `network_summary.csv` | CSV with header | Per-network Table 1 statistics |
